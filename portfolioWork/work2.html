Javascript Project Pools

The project pool submissions were extremely important because Javascript is very different from HTML or CSS as it is a programming language itself. It has several frameworks and libraries and the very basis of website interactivity ( excluding animations and transforms done through CSS ) stands in Javascript. Unfortunately, I do believe that for understanding Javascript you should have previous knowledge in other programming languages as C# or C++ and a lot of exercise behind.

03.03.03 Project Pool 1

A small interactive piece of work, “Smack the Fly” , put my Javascript knowledge to test, to be more precise connecting audio with events that set attributes and remove classes as well as letting the fly randomly move across the screen through the Math Javascript object.

03.03.04 Project Pool 2

The “Random SVG” was a bit more challenging as it has thought me how to create elements through a unique identifier of the namespace. It was really useful to figure out how I can set the attribute of the inner width of the elements randomly and how to move them around through a for loop. All in all, I have had the chance to learn in a more detailed manner about how Javascript actually works.
