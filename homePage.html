Adaptability

This website was created with the intent to serve as a canvas on which I can project my work that encapsulates the assignments done in the first semester of the Multimedia Design course at KEA Academy including the following themes: Basic Web, Basic Content, Basic Animation and Basic UX.

In creating this presentation website I was inspired by the Italian Renaissance art period which I combined with a more modern style, Deconstruction. The typography was carefully chosen so that there is enough visual difference between the fonts to serve their own purpose and they can still match together very well.

For each page I have chosen a keyword-title as I tried to break the norms and name each page in an unique way that talks more to the user than a classic boring name as in the navigation bar.
